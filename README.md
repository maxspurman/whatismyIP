# 🌐 What Is My IP

A simple web app that shows publicly visible connection and system information.


## 🚀 Features

* Detects your public IPv4 address
* Detects your IPv6 address if available
* Shows operating system and version
* Shows browser and version
* Displays country and city
* No personal data is stored

## 🛠 Tech Stack

* Python (Flask)
* HTML / CSS / JavaScript

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py

Open in browser:

```
http://127.0.0.1:5000
```

## ⚠️ Disclaimer

This tool is for educational purposes only.
Location and network details are based on publicly visible IP data and may not always be exact.
